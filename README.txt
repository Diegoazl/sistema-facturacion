
SISTEMA DE FACTURACIÓN - PHP OOP + MVC

Este sistema incluye:

- CRUD completo para Clientes, Productos, Facturas, Vendedores, Empresas.
- Módulo para agregar productos por factura con cálculo de totales.
- Estilo moderno con CSS personalizado.
- Generación de PDF para facturas (requiere DomPDF).
- Base de datos incluida: sistema_facturacion.sql

Estructura:
- config/ -> conexión BD
- models/ -> lógica de negocio
- controllers/ -> flujo de datos
- views/ -> formularios y tablas CRUD
- css/ -> estilos
- pdf/ -> exportar factura a PDF
